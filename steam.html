GetFreeplay

GetFreeplayをダウンロード

GetFreeplayとは？

Steamというのは、主にPCゲームを取り扱うオンラインゲームストアのこと。イメージとしてはPCゲーム版のAppstoreやGoogleplaystore。

Steamでは有料販売されているゲームタイトルを期間限定で無料プレイできることがある。
しかし、そのタイトルを毎回調べるのは面倒。そこでこのソフト。このソフトは自動的に期間限定の無料タイトルを調べてユーザに表示する。

GetFreeplayができること

起動するだけで、現在フリープレイ対象のゲームを表示します。

なぜ作った？

やりたかったゲームのフリープレイがあるってことを、フリープレイ期間終了2時間前に知ったから。

詳細

言語 : Python3

使用ライブラリ : requests, BeautifulSoup, re, webbrowser

Source Code


import requests
from bs4 import BeautifulSoup
import re
import webbrowser

URL = ('https://nogameb.com/archives/steam-play-for-free.html')
res = requests.get(URL)
soup = BeautifulSoup(res.text, "html.parser")

free_game = soup.find('span', id = 'toc1') 
free_game2 = soup.find('span', id = 'toc2')

f = str(free_game.get_text()) 
f2 = str(free_game2.get_text())
print(f + '\n' + f2 + '\n情報が送れている場合などがあります。Enterを押すとページに飛べます')# + '\n' + f3 + '\n' + f4 + '\n' + f5 + '\n' + f6 + '\n' +f7

URL__ = 'https://steamdb.info/upcoming/free/'

input()
webbrowser.open(URL__)
