Alexaランキング取得

Alexaランキング取得プログラムをダウンロード

.exe形式のファイルをダウンロードすると
セキュリティソフトがウイルスと誤認して
強制削除してしまいます。
事前にセキュリティソフトを一時停止するか、
インストールするフォルダーを
ウイルス診断から除外してください。
フォルダーをセキュリティソフトから除外する方法 理解してAlexaランキング取得(exe)をダウンロード

Alexaランキングとは

ウェブサイトの利用状況に関するデータを集め、ウェブサイトがどれだけの人に
視聴されているかを調査している。alexa.comでは、ドメインへの1日当たりのアクセス数（トラフィック）や訪問者数、直帰率、世界および国内におけるランキングなどを確認できる。

出典：Wikipedia アレクサ・インターネット より

要するに、どれだけそのウェブサイトがどれだけアクセスされているかが大体わかるサイトだ。

何ができる？

サイトのURLを入力するとAlexaランキングを教えてくれる。alexa.comに行ってURLを入力するより早くて簡単。

詳細

言語 : Python3

使用ライブラリ : requests, BeautifulSoup, re

Source Code(対話)


import requests
from bs4 import BeautifulSoup
import re
import pyperclip

#This is AUTO version

def alexa_rank():
    want_URL = (input('please URL : '))  

    URL = ('https://www.alexa.com/siteinfo/' + want_URL)
    res = requests.get(URL)
    print('loading...')
    soup = BeautifulSoup(res.text, "html.parser")

    ranking = soup.find('p', class_='big data') #big dateというclassにランキング情報in
    ranking = re.sub('[
 # 
]', ' ', ranking.text) #ここでspanなど不要な情報を削除


    print(want_URL + '\n    global rank #' + ranking.strip())#stripでスペースと改行を削除

    Source = ('\nFor more :' + URL + '\n\n')
    print(Source)

while True:
    alexa_rank()
